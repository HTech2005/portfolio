README - Template Portfolio Étudiant / Développeur
Présentation

Bienvenue !
Ce template est un portfolio moderne et responsive, idéal pour les étudiants, développeurs ou designers souhaitant présenter leurs compétences, projets et services en ligne.

Il est construit en HTML, CSS et JavaScript, et est facilement personnalisable. Il inclut des sections :

Accueil (Hero)

À propos

Services

Portfolio / Projets

Contact

Footer avec réseaux sociaux

Effets visuels et animations (préloader, barres de compétences, overlay portfolio, scroll to top, etc.)

Aperçu du Template

Design moderne et épuré

Responsive : fonctionne sur mobiles, tablettes et desktops

Animations et effets visuels : préloader, particules, overlay projets, barres de compétences animées

Facile à personnaliser : images, textes, couleurs et icônes

Optimisé SEO et UX

Instructions d’utilisation

Télécharger le package et extraire tous les fichiers dans un dossier.

Ouvrir index.html dans un navigateur pour visualiser le site.

Modifier le contenu :

Texte : dans les sections À propos, Services, Portfolio, Contact.

Images : remplacer celles de images/ ou les URLs existantes.

Couleurs : dans css/style.css, modifier les variables CSS :root.

Ajouter vos projets : dupliquer un bloc .portfolio-item et mettre vos informations et images.

Personnaliser les icônes : Font Awesome est déjà inclus pour changer les icônes selon vos besoins.

Formulaire de contact : le formulaire HTML est prêt, mais pour envoyer les emails, configurez votre backend (PHP, Node.js, ou service externe comme Formspree).

Personnalisation rapide

Couleurs principales :

:root {
    --primary: #6c5ce7;
    --secondary: #a29bfe;
    --accent: #fd79a8;
    --dark: #2d3436;
    --light: #dfe6e9;
}


Polices : Poppins, via Google Fonts (déjà intégré).

Icônes : Font Awesome 6.4 (déjà inclus via CDN).
